!!!Файл icon.png обязательно должен быть в папке с исполняемым файлом NicknameFinder.exe!!!

Если не запускается:
1. Установить Python на OS
2. Установить библиотеки httpx, requests, tkinter
	pip install httpx;
	pip install requests;
	pip install tkinter.
3. Снова попробовать запустить 
4. Если на запускается программа, открыть через Python IDLE файл main.py из папки Python
